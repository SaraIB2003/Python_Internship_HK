
# Python Internship Tasks - HK Software House

These are my tasks for the Python Developer internship at HK Software House & Training Center.

## Task 1 - Contact Management System
A CLI app to manage contacts. Supports add, search, filter, update, delete, export to CSV and more.
All data is saved in a JSON file.

→ folder: task1_contact_manager

## Task 2 - Multi User Auth System
A CLI login and registration system with password hashing, account lockout and admin panel.
Passwords are stored as SHA-256 hashes, never plain text.

→ folder: task2_auth_system

## How to run

Task 1:
  cd task1_contact_manager
  python contacts.py

Task 2:
  cd task2_auth_system
  python auth.py

## Tech used
- Python 3
- JSON for data storage
- hashlib for password hashing
- Standard library only, no extra installs needed
